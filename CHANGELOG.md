# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-03-06

### Added

- Initial project setup (`package.json`, `tsconfig.json`, `eslint`, `prettier`, `vitest`).
- Implementation of OpenSearch API client for CiNii Research (Phase 1 to Phase 4).
- JSDoc annotations and usage examples.
- Schema and specification documents.
- Support for NPM package publishing, including `exports` map and definition generation (`index.d.ts`).

### Fixed

- Fixed linting and formatting issues across the codebase.
- Revised type checking configuration to strictly ensure quality.
